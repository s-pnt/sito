Liste

In questo capitolo vedremo le liste, iniziamo come al solito a crearne una nella shell: 

>>> città = ['Milano', 'Napoli', 'Torino', 'Roma', 'Palermo']

Vediamo la lista di città ed il tipo: 

>>> città
['Milano', 'Napoli', 'Torino', 'Roma', 'Palermo']
>>> type(città)
<class 'list'>

Python ovviamente ci suggerisce che questa è una lista ma se osserviamo bene non è altro che una lista di stringhe. Proviamo ad accedere ai vari elementi della lista tramite l'indice, come abbiamo fatto con le stringhe, per vedere di che tipo è l'elemento Napoli, per esempio: 

>>> type(città[1])
<class 'str'>

Infatti ci dice che è una stringa. Ma una lista può contenere solo stringhe? 

>>> città_numeri = ['Roma', 512, 'Catania', 'Milano', -12.6, 0]
>>> città_numeri
['Roma', 512, 'Catania', 'Milano', -12.6, 0]
>>> type(città_numeri)
<class 'list'>

Abbiamo creato un'altra lista ma stavolta contenente stringhe, numeri positivi, negativi, interi e non, quindi una lista può contenere diversi tipi di dati.

Definizione: Una lista è un insieme ordinato di diversi tipi di dati. Ora vediamo se possiamo usare i metodi utilizzati in precedenza sulle stringhe anche sulle liste: 

>>> città_numeri[2]
Catania
>>> città_numeri[-1]
0
>>> città_numeri[3:]
['Milano', -12.6, 0]
>>> len(città_numeri)
6

Questo è un altro esempio di flessibilità di Python, abbiamo usato il metodo di affettamento sia sulle liste che sulle stringhe. A proposito di flessibilità, cosa succede se proviamo a costruire una lista di liste? Proviamo a crearne una di esempio su cui fare qualche prova: 

>>> persona = [['Mario', 'Rossi'], ['Corso Garibaldi, 21', 'Milano', 10100], ['Celibe', 'Impiegato']]
>>> persona
[['Mario', 'Rossi'], ['Corso Garibaldi, 21', 'Milano', 10100], ['Celibe', 'Impiegato']]

La lista persona che abbiamo appena creato contiene a sua volta delle liste, avremmo ottenuto la stessa struttura di dati dichiarandola come segue: 

>>> persona = [['Mario', 'Rossi'],
...            ['Corso Garibaldi, 21', 'Milano', 10100],
...            ['Celibe', 'Impiegato']]

Così è più facile realizzare che abbiamo appena creato una matrice, cioè una tabella da tre righe contenenti a loro volta due o più elementi, una struttura dati di questo tipo:

Indice 0 1 2
0 Mario Rossi
1 Corso Garibaldi, 21 Milano 10100
2 Celibe Impiegato

L'impatto delle matrici nei più svariati ambiti dell'informatica è enorme, basti pensare che tutte le immagini digitali non sono altro che matrici contenenti numeri per indicare il colore di ogni pixel. Inoltre ogniqualvolta si ha a che fare con una grossa quantità di dati, al fine dell'analisi o della visualizzazione in grafi o tabelle, sarà necessario raccoglierli e organizzarli dentro a delle matrici. Ora vediamo come accedere ai vari elementi: 

>>> persona[0]
['Mario', 'Rossi']
>>> persona[0:]
[['Mario', 'Rossi'], ['Corso Garibaldi, 21', 'Milano', 10100], ['Celibe', 'Impiegato']]
>>> persona[0][0]
Mario
>>> persona[0][1]
Rossi
>>> persona[1][0]
Corso Garibaldi, 21
>>> persona[2][1]
Impiegato
>>>

Prendiamo come riferimento la tabella poco sopra che rappresenta la nostra matrice e l'ultima riga, persona[2][1], stiamo chiedendo a Python di mostrarci l'elemento della riga 2 e colonna 1. Che succede invece se richiediamo un elemento che non c'è? Proviamo! 

>>> persona[2][3]
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
IndexError: list index out of range
>>>

L'ultima riga parla chiaro, Python ci dice che c'è un errore di indice e che l'indice della lista è fuori dal range.
Vedremo più avanti come gestire gli errori, per intanto si rende utile il metodo len() che abbiamo usato prima sia sulle liste che sulle stringhe, che ci permette di accedere a tutti gli elementi senza ottenere errori di questo tipo.
Adesso vediamo alcuni importanti metodi per manipolare le liste, iniziando da append() che, come suggerisce la parola stessa, ci permette di aggiungere elementi in fondo alla lista: 

>>> persona = ['Mario, 'Rossi']
>>> persona.append('Tecnico')
>>> persona
['Mario', 'Rossi', 'Tecnico']

Il metodo pop(i) ci restituisce il valore della lista all'indice i dopodiché lo elimina, vediamolo all'opera con la lista appena creata: 

>>> persona.pop(2)
'Tecnico'
>>> persona
['Mario, 'Rossi']

Ha restituito Tecnico che era l'elemento 2 della lista e l'ha eliminato, come ci aspettavamo. Il metodo pop() può essere utilizzato senza argomenti, in questo caso restituisce ed elimina l'ultimo elemento della lista: 

>>> persona.pop()
'Rossi'
>>> persona
['Mario']

Il metodo extend() ci torna utile quando vogliamo aggiungere più elementi alla nostra lista, se utilizziamo append() come fatto in precedenza, viene creata una sottolista all'interno della lista principale e non è sempre il comportamento desiderato, facciamo qualche esempio per essere più chiari: 

>>> lista_1 = ['Mario', 'Rossi']
>>> lista_2 = ['Impiegato', 32]
>>> lista_1.append(lista_2)
>>> lista_1
['Mario', 'Rossi', ['Impiegato', 32]]

Ora che abbiamo visto il comportamento di append() quando si aggiungono più elementi, vediamo invece extend(): 

>>> lista_1 = ['Mario', 'Rossi']
>>> lista_2 = ['Impiegato', 32]
>>> lista_1.extend(lista_2)
>>> lista_1
['Mario', 'Rossi', 'Impiegato', 32]

E ancora cosa succede se usiamo semplicemente il +: 

>>> lista_1 = ['Mario', 'Rossi']
>>> lista_2 = ['Impiegato', 32]
>>> lista_1 + lista_2
['Mario', 'Rossi', 'Impiegato', 32]

Infine vediamo il metodo remove() che ci permette di eliminare un elemento dalla lista specificandone il valore: 

>>> persona = ['Mario', 'Rossi', 'Impiegato']
>>> persona.remove('Rossi')
>>> persona
['Mario', 'Impiegato']
