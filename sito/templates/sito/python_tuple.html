Tuple

tupla = ('sono', 'una', 'tupla')

Definizione: una tupla è una lista immutabile.

Le tuple hanno molti contesti di utilizzo, in primo luogo sono più veloci delle liste. Inoltre è buona norma usarle quando abbiamo la certezza che i nostri dati non devono cambiare in quanto, essendo immutabili, non potremo modificarle nemmeno per sbaglio.
Vediamo come funzionano: 

>>> tupla = ('le', 'tuple', 'sono', 'immutabili')
>>> tupla[2]
'sono'
>>> tupla[-1]
'immutabili'
>>> tupla[1:4]
('tuple', 'sono', 'immutabili')

Come vediamo, i metodi di affettamento e indicizzazione funzionano anche con le tuple. Proviamo ora a modificarla: 

>>> tupla[0] = 'modifica'
Traceback (most recent call last):
  File "<stdin>", line 1, in <module>
TypeError: 'tuple' object does not support item assignment

Python è chiarissimo: la nostra 'tuple' non supporta l'assegnazione, quindi non può essere modificata.
Proviamo a modificarla in maniera diversa: 

>>> tupla = ('prova', 'modifica')
>>> tupla
('prova', 'modifica')

Può sembrare che abbiamo modificato la tupla ma se osserviamo bene in realtà non è rimasta traccia della precedente ('le', 'tuple', 'sono', 'immutabili'), ne abbiamo semplicemente creata una nuova, andando a sovrascrivere la precedente.
