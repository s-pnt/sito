Variabili

Iniziamo a farci un'idea di cosa sono e come funzionano le variabili in Python.
Dichiariamo una variabile di nome x e assegnamole un valore di 10. Per questo esempio abbiamo usato il nome x ma avremmo potuto usare qualunque nome, una convenzione che si usa nella scelta dei nomi delle variabili è che sia esplicativo rispetto alla funzione della variabile stessa, ad esempio in questo caso avremmo potuto chiamare la variabile numero in quanto le assegnamo un valore numerico, comunque teniamo x e andiamo avanti: 

>>> x = 10

Definizione: una variabile in informatica è un contenitore di dati situato in una porzione di memoria destinata a contenere valori, suscettibili di modifica nel corso dell'esecuzione di un programma. Una variabile è caratterizzata da un nome.

E' esattamente quello che abbiamo fatto qui sopra, abbiamo scelto il nome x e le abbiamo fatto contenere un dato numerico.
Vediamo la nostra x, fintanto che usiamo la shell interattiva non sarà necessario usare print(). 

>>> x
10

Ora, per curiosità, proviamo a vedere di che tipo è la variabile x che abbiamo appena creato, noi abbiamo indicato solo un valore (10) ma non abbiamo specificato se fosse un intero, una stringa di testo o un altro tipo di dato. Per visualizzare il tipo di dato sarà sufficiente usare la funzione type() già presente in Python: 

>>> type(x)
<class 'int'>

Abbiamo chiesto a Python che tipo di dato è la variabile x e abbiamo ottenuto come risultato class 'int', cioè una classe di tipo int (intero), tralasciamo per ora il concetto di classe che tratteremo più avanti, chiaramente Python ci suggerisce che x è un numero intero. Potrebbe sembrare un comportamento normale ma in altri linguaggi di programmazione la semplice dichiarazione e inizializzazione di una variabile non è così immediata, richiede una maggiore quantità di righe di codice, che di conseguenza si traduce in più errori e maggiore difficoltà di manutenzione e lettura dell'intero codice. Ma andiamo avanti, proviamo a fregare Python! 

>>> x = x + 0.25

Qui stiamo chiedendo a Python di aggiornare la variabile x e di farla diventare uguale a x + 0.25. Visualizziamo la variabile x: 

>>> x
10.25

Ma come? Era un intero, ora ha la virgola! Verifichiamo il tipo di x: 

>>> type(x)
<class 'float'>

Ci ha fregati Python, la variabile x è diventata un float, cioè un numero a virgola mobile. Giochiamo ancora un pò con le variabili: 

>>> x = 'ciao mondo!'
>>> x
ciao mondo!

Come abbiamo creato una variabile numerica, l'abbiamo modificata in una di tipo testuale, vediamo di che tipo: 

>>> type(x)
<class 'str'>

Ora è di tipo 'str', cioè stringa. Questa flessibilità non è una cosa comune a tutti i linguaggi di programmazione ma è un'arma a doppio taglio, da un lato ci permette di lavorare su diversi tipi di dati senza preoccuparci di scomode dichiarazioni e conversioni, dall'altro ci mette in una posizione in cui dobbiamo controllare i dati che vengono inseriti, per esempio poniamo di lavorare sul software di contabilità di un'azienda, dovremmo preoccuparci che non vengano inserite lettere o simboli quando ci aspettiamo degli importi, per evitare problemi di funzionamento del programma stesso o persino di corrompere i dati presenti nel sistema.
Vediamo un ultimo tipo, si chiamano variabili booleane e possono contenere solo due valori, vero o falso: 

>>> x = True
>>> x
True
>>> type(x)
<class 'bool'>
>>> x = False
>>> x
False

Proviamo qualcosa di un pò più complesso sempre con le variabili booleane, di seguito assegnamo ad x il risultato di una comparazione, nello specifico chiediamo se 12 è maggiore di 2:

>>> x = 12 > 2
>>> x
True

Ora abbiamo una panoramica chiara del concetto di variabile e dei diversi tipi, nel prossimo capitolo vedremo approfondiremo le stringhe a cui abbiamo accennato poco sopra.
