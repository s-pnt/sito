Script Python

Apriamo l'editor di testo (PyCharm, Notepad++, Sublime text o quello che preferite) e scriviamo l'avanzatissimo programma che abbiamo visto nel capitolo precedente: 

print('Hello, world!')

Salviamolo con estensione .py, ad esempio prova.py, questa è l'estensione che indica al sistema che il file in questione è uno script Python. Aprite la riga di comando e spostatevi dove avete salvato il programma, in Windows un comodo trucchetto è di tenere premuto maiuscolo e cliccare il tasto destro dove avete salvato il file .py, in questo modo nel menù a tendina compare la voce Apri la finestra di comando qui che apre il prompt direttamente in quella cartella, altrimenti spostiamoci tra le cartelle di sistema con il comando cd ed eseguiamo il nostro programma:

Windows 

python prova.py
Hello, world!

Linux/Mac 

python3 prova.py
Hello, world!

Ecco fatto, il nostro programma è già pronto ad essere eseguito senza la necessità di compilarlo e funziona su ogni piattaforma su cui sia installato Python.
