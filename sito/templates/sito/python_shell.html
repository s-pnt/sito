Shell interattiva

Abbiamo visto nel capitolo precedente come installare Python e verificare che l'installazione fosse andata a buon fine, per farlo abbiamo lanciato una shell interattiva. Vediamo cos'è, da Linux apriamo la shell e digitiamo python3, su Windows invece apriamo il prompt dei comandi e scriviamo python, dovremmo ricevere un output simile al seguente, con qualche differenza in base al sistema operativo su cui lo eseguiamo: 

Python 3.7.0 (v3.7.0:1bf9cc5093, Jun 27 2018, 04:06:47) [MSC v.1914 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

Il simbolo >>> ci indica che la shell interattiva è pronta a ricevere istruzioni, in questo modo possiamo interagire con Python riga per riga, senza preoccuparci di scrivere un intero programma e poi eseguirlo, ma limitandoci ad osservarne il comportamento ad ogni istruzione. Il primo programma che si impara in ogni linguaggio di programmazione è il famoso 'Hello, world!': 

>>> print('Hello, world!')
Hello, world!

L'istruzione print() indica a Python di "stampare" (mostrare a schermo) ciò che è contenuto tra le parentesi, in questo caso la stringa 'Hello, world!'. Niente di più semplice, vero? Eppure Python va oltre! 

>>> 'Hello, world!'
Hello, world!

Wow! Fintanto che usiamo la shell interattiva non è necessario scrivere print, è sufficiente scrivere una stringa o un numero e verrà mostrato: 

>>> 21
21
>>> 2 + 5
7
>>> 3 * 2 + 5
11

Abbiamo già una calcolatrice pronta all'uso ma come si comporta Python se cambiano la posizione degli operatori? 

>>> 5 + 3 * 2
11

Ok, Python rispetta la priorità degli operatori e quello che fa dietro le quinte è trasformarla come faremmo matematicamente, cioè 5 + ( 3 * 2 ). Torniamo alla nostra shell interattiva: 

>>> _
11

Il trattino basso è una variabile che crea Python e contiene ogni volta l'ultimo valore che abbiamo mostrato come output: 

>>> _ + 4
15
>>> _ * 2
30

Per uscire dalla shell interattiva ci sono diverse scorciatoie da tastiera in base al sistema operativo (Ctrl-C, Ctrl-D, Ctrl-Z), oppure è sufficiente digitare exit(). Ora che abbiamo un pò di confidenza con la shell interattiva vedremo come scrivere e salvare i nostri primi programmi.
