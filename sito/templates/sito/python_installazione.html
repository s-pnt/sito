Installazione

Windows

Download Python 3.7

Linux

Di solito su Linux è preinstallato in quanto parte delle funzionalità del sistema operativo stesso sono scritte in Python. Per verificare se Python è installato è sufficiente digitare python3 da riga di comando, se non fosse già installato sarà sufficiente digitare:

sudo apt-get install python3.7

Mac

Di solito anche su Mac, che è un sistema operativo basato su Linux, Python è preinstallato.

Per verificarlo è sufficiente digitare python3 da riga di comando, altrimenti installare la versione appropriata tra le seguenti:

Per Mac OS X 10.9 e successivi: macOS 64-bit installer

Per Mac OS X 10.6 e successivi: macOS 64-bit/32-bit installer



Per verificare che l'installazione sia andata a buon fine è sufficiente aprire la shell dei comandi e digitare python in caso di Windows o python3 in caso di Linux/Mac. Dovremmo ricevere un output simile al seguente:

Python 3.7.0 (v3.7.0, Jun 27 2018, 04:06:47) [MSC v.1914 32 bit (Intel)] on win32
Type "help", "copyright", "credits" or "license" for more information.
>>>

A questo punto consiglio di installare un IDE, cioè un comodo editor con interfaccia grafica, nato per progetti in Python, in quanto ci agevolerà nel lavoro con funzioni di autocompletamento, autoindentazione, ci segnalerà errori e altre utili funzioni. L'IDLE che viene installato insieme a Python è già un ottimo strumento, io per preferenza personale consiglio PyCharm Community Edition, è free, veloce, funziona su Windows, Mac e Linux ed è adatto a diversi tipi di progetti.
